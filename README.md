## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Node 
- Npm 
- Composer 
- Text editor (VsCode)

### Install

1. Clone the repo
   ```sh
   git clone git@github.com:Dipfella/Portafolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
